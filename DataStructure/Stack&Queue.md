# Stack
- LIFO(Last In First Out) : 스택은 가장 마지막으로 들어간 데이터가 가장 첫 번째로 나오는 성질을 가진 자료 구조
- 재귀적인 함수, 알고리즘에 사용
- 웹 브라우저 방문 기록 등에 쓰임
- 삽입 및 삭제 O(1), 탐색 O(n)
# Queue
- FIFO(First In First Out) : 큐는 먼저 집어넣은 데이터가 먼저 나오는 성질을 지닌 자료 구조
- 삽입 및 삭제 O(1), 탐색 O(n)
- CPU 작업을 기다리는 프로세스, 스레드 행렬 또는 네트워크 접속을 기다리는 행렬, 너비 우선 탐색, 캐시 등에 사용
